Project 3: Serverbeheer Intro

Korte omschrijving

Het olod Serverbeheer Intro behandelt de basisvaardigheden van systeembeheer via de mogelijkheden van een eerste netwerkbesturingssysteem, namelijk Windows.

Het olod Serverbeheer Intro bereidt voor op het olod Serverbeheer.

Projectdetails

In dit project werd een eerste Windows-serveromgeving opgezet en getest, inclusief gebruikersbeheer, basis netwerkconfiguratie en bestandsdeling binnen het netwerk.

Er werd aandacht besteed aan het correct configureren van rollen en functies van de server, zodat een veilige en werkende netwerkopstelling ontstaat.

  • Een Windows-serveromgeving correct opzetten
  • Gebruikersbeheer en permissies configureren
  • Netwerkrollen en bestandsdeling begrijpen en toepassen
  • Basis netwerkconfiguratie uitvoeren voor server en clients
  • Voorbereiding op verder Serverbeheer olod

  • Windows Server
  • Active Directory
  • Bestandsdeling en permissies
  • Netwerkconfiguratie tools
