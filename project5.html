Project 5: Besturingssystemen

Korte omschrijving

In het olod besturingssystemen wordt de werking van computersystemen aangeleerd. De student past alle leerinhouden toe in een Linux omgeving.

Het olod besturingssystemen bereidt voor op alle olods uit de leerlijn systeembeheer.

Projectdetails

Voor dit project werd een SSH-sleutel gegenereerd waarmee er zonder wachtwoord kan ingelogd worden op een andere machine binnen het netwerk.

Met behulp van een SSH-configuratiebestand kan er eenvoudig verbinding gemaakt worden met deze machine via een korte, zelfgekozen naam.

  • SSH-sleutels genereren voor veilige, wachtwoordloze authenticatie
  • SSH gebruiken om verbinding te maken met een andere machine
  • Een SSH-configuratiebestand opstellen om verbindingen te vereenvoudigen
  • Begrijpen hoe meerdere machines met dezelfde login te beheren

  • SSH (Secure Shell)
  • SSH keygen
  • SSH-configuratiebestand
  • Linux command line
