Project 4

Project 4: Webtechnologie

Korte omschrijving

Het olod Webtechnologie behandelt het creëren van interactieve en grafische websites met behulp van HTML en CSS. Dit olod bereidt voor op het bouwen van een volledige statische website.

We leerden hoe je Bootstrap kan gebruiken om snel een responsieve layout te maken, maar ook hoe alles werkt met JavaScript en CSS, zodat je de werking achter de schermen begrijpt. Daarnaast werd aandacht besteed aan semantiek, structuur, toegankelijkheid en lay-out van websites.

Projectdetails

Voor dit project heb ik een website gemaakt met HTML, CSS en JavaScript. Ik heb Bootstrap gebruikt voor een snelle, responsieve layout en tegelijkertijd geleerd hoe de componenten intern werken.

Er is speciale aandacht besteed aan semantiek en structuur, zodat de site toegankelijk is voor alle gebruikers. CSS en JavaScript werden ingezet om de lay-out en interactiviteit van de website te optimaliseren.

  • Begrijpen hoe HTML, CSS en JavaScript samenwerken
  • Responsieve websites bouwen met Bootstrap
  • Semantiek en structuur toepassen voor toegankelijkheid
  • Interactiviteit en layout optimaliseren met CSS en JS
  • Het gebruik van externe frameworks begrijpen en toepassen
